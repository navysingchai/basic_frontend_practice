Exercise 7: Removing Duplicates

ใช้วิธีสร้าง function in_array(item, arr) ซึ่งจะ return ว่า item ที่ใส่ไป มีกี่ตัว ถ้ามี มันจะ return เป็น array ของ item นั้น ๆ ที่ซ้ำ
จากนั้นใน remove_duplicate ก็ไปเรียกใช้ in_array แล้วเช็ค in_array().length !== 1 หรือไม่ ถ้าไม่เท่ากับ 1 แสดงว่ามันต้องมากกว่า ก็คือให้ลบได้เลย
